Termos de Uso

1. Aceitação dos Termos

Ao criar seu RG da Saúde na plataforma Vitae, você concorda com estes Termos de Uso. A Vitae é uma plataforma digital de gestão de dados pessoais de saúde.

2. Dados Coletados

Coletamos: nome, CPF, data de nascimento, dados de saúde (alergias, medicamentos, cirurgias, exames), contato de emergência e plano de saúde. Todos os dados são fornecidos voluntariamente pelo usuário.

3. Armazenamento e Segurança

Seus dados são armazenados com criptografia de ponta a ponta. Utilizamos servidores seguros e seguimos as melhores práticas de segurança da informação.

4. Compartilhamento de Dados

Seus dados nunca são compartilhados sem sua autorização expressa. Cada compartilhamento (via QR Code ou pré-consulta) requer uma nova autorização, registrada com data, hora e destinatário.

5. Seus Direitos

Conforme a LGPD, você tem direito a: acessar, corrigir, excluir e exportar seus dados a qualquer momento. Para exercer esses direitos, acesse as configurações do app ou entre em contato conosco.

6. Responsabilidades

Você é responsável pela veracidade dos dados informados. A Vitae não substitui orientação médica profissional. Os dados e resumos gerados por IA são ferramentas de apoio.

7. Pagamento

O RG da Saúde tem custo único de R$4,99. Não há cobranças recorrentes para o paciente.

‹
Termos Profissional

1. Uso Profissional

A plataforma Vitae para Profissionais é destinada exclusivamente a médicos e profissionais de saúde devidamente registrados em seus conselhos (CRM, CRO, etc).

2. Responsabilidades do Profissional

O profissional se compromete a: utilizar os dados exclusivamente para atendimento médico; não compartilhar dados com terceiros sem autorização do paciente; manter sigilo profissional.

3. Dados do Paciente

Os dados recebidos via pré-consulta pertencem ao paciente. O acesso é concedido mediante consentimento expresso do titular, conforme a LGPD (Art. 7º, I).

4. One Minute Summary

Os resumos gerados por IA são ferramentas auxiliares. Não substituem a anamnese clínica. O profissional é responsável por suas decisões clínicas.

5. Plano e Limites

O plano básico permite até 100 pacientes/mês por R$19,90. Pacientes adicionais podem ser adquiridos em pacotes.

6. LGPD

A Vitae atua como operador dos dados; o profissional atua como controlador conjunto. Ambos compartilham responsabilidades conforme a LGPD.

Política de Privacidade

1. Controlador dos Dados

Vitae Health Tecnologia Ltda.
Encarregado (DPO): privacidade@vitae.app

2. Dados Pessoais Coletados

Dados cadastrais: nome, CPF, data de nascimento, sexo, email, telefone.
Dados de saúde (sensíveis): tipo sanguíneo, peso, altura, alergias, medicamentos, cirurgias, condições crônicas, exames laboratoriais, plano de saúde.

3. Base Legal

O tratamento dos dados pessoais é baseado no consentimento do titular (Art. 7º, I e Art. 11, I da LGPD). Dados sensíveis de saúde são tratados com base no consentimento específico e destacado.

4. Finalidade

Seus dados são utilizados para: criação do RG da Saúde; compartilhamento autorizado com profissionais de saúde; geração de resumos de pré-consulta; lembretes de exames.

5. Compartilhamento

Dados são compartilhados somente com profissionais de saúde mediante consentimento expresso a cada compartilhamento. Cada autorização é registrada com data, hora e destinatário.

6. Armazenamento

Dados armazenados com criptografia AES-256. Servidores com certificação ISO 27001. Backups criptografados.

7. Seus Direitos (Art. 18 da LGPD)

Você tem direito a: confirmação do tratamento; acesso aos dados; correção de dados incompletos; anonimização ou eliminação; portabilidade; revogação do consentimento.

8. Exclusão de Dados

Você pode solicitar a exclusão total dos seus dados a qualquer momento. A exclusão é irreversível e será processada em até 15 dias úteis.

9. Contato

Para exercer seus direitos: privacidade@vitae.app
Horário: segunda a sexta, 9h às 18h.
